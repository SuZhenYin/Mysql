 5. 掌握基本的Lunix常用命令 
 
   命令一： sudo xrandr -s 10,作用是将屏幕最大化
   
   命令二： cd ,作用是进入命令
   
   命令三： sudo apt-ge0t install 软件名，作用是下载软件
   
   命令四： mkdir+文件夹名，作用是创建文件夹
   
   命令五： vi+文件名.后缀， 作用是创建文件
   
   命令六： rm+文件名，作用是删除文件名
   
   命令七： pwd,作用是显示当前目录下的所有的文件和文件夹
